&#x20; Version History / 项目版本演进



本项目 `industry-research-radar` 是从原项目 `Global\_QDII\_Risk\_Monitor` 中拆分出来的独立行业研究模块。当前 GitHub 仓库的 Git 历史从拆分节点开始，但项目功能本身经历了从 V1 到 V5.3 的逐步演进。



当前仓库包含最新可运行版本代码，但不包含真实缓存、报告、日志、备份、私有配置和历史 data 文件。







&#x20;  V1：行业搜索原型



V1 阶段建立了单行业搜索入口，支持通过行业名称查询基础行业框架。



主要功能包括：



&#x20;支持中英文行业别名；

&#x20;输出行业阶段、核心驱动因素、风险信号和产业链；

&#x20;从“输入行业名”快速生成基础行业分析框架。



目标是先完成最小可用的行业趋势查询原型。



&#x20;



&#x20;  V2：行业框架扩展



V2 阶段重点扩展行业覆盖范围和中文解释能力。



主要功能包括：



&#x20;扩展行业池；

&#x20;增加更多子行业、催化因素和风险信号；

&#x20;强化中文字段和行业解释；

&#x20;提高行业页面的可读性和覆盖面。



目标是让系统从单一行业工具变成更完整的行业趋势雷达。



&#x20;



&#x20;  V3：市场价格确认



V3 阶段引入市场价格代理数据，用价格行为验证行业热度。



主要功能包括：



&#x20;引入 MA20 / MA50 / MA200 等价格确认指标；

&#x20;增加 1M / 3M / 6M 表现；

&#x20;增加相对强弱指标；

&#x20;形成市场价格确认分。



目标是避免只看概念热度，而是加入市场行为验证。



&#x20;



&#x20;  V4：新闻催化与动态快照



V4 阶段开始引入新闻催化、动态快照和框架复核。



主要功能包括：



&#x20;增加新闻缓存；

&#x20;接入 Google News RSS / Yahoo Finance 等新闻来源；

&#x20;增加新闻相关性判断；

&#x20;区分 high / medium / low relevance 新闻；

&#x20;增加催化框架；

&#x20;增加动态催化快照；

&#x20;增加框架复核；

&#x20;增加多种 audit 脚本。



目标是判断行业热度是否有真实新闻和催化事件支撑。



&#x20;



&#x20;  V4.13 - V4.15：催化语义修复



V4.13 到 V4.15 阶段主要修复催化框架和新闻语义问题。



主要改动包括：



&#x20;修复框架 override 展示问题；

&#x20;修复多源新闻合并排序；

&#x20;修复 catalyst event 语义；

&#x20;修复白酒、黄金、AI 算力等行业的类型和影响方向误判；

&#x20;减少 `technology / positive / high` 这类模板化误判。



目标是提高催化事件解释的可信度和行业适配性。



&#x20;



&#x20;  V5.1：本地研究摘要



V5.1 阶段新增本地规则版研究摘要。



该模块基于趋势评分、价格确认、催化热度、缓存新闻和行业框架，生成系统初判。



主要输出包括：



&#x20;趋势健康度；

&#x20;健康度解读；

&#x20;价格与新闻是否共振；

&#x20;共振解读；

&#x20;当前催化主线；

&#x20;风险信号；

&#x20;待验证问题；

&#x20;人工复核建议。



V5.1 不是严格意义上的 AI Agent，而是本地规则版研究摘要生成器。



&#x20;



&#x20;  V5.2：AI 研究输入包 / ChatGPT Prompt 导出器



V5.2 阶段没有接入 API，而是生成可复制给 ChatGPT 的结构化研究 Prompt。



主要功能包括：



&#x20;不接 OpenAI API；

&#x20;不调用模型；

&#x20;不读取 API key；

&#x20;生成 ChatGPT 研究输入包；

&#x20;包含 V5.1 摘要、高相关新闻、催化主线、风险信号、待验证问题和数据说明；

&#x20;支持下载 Markdown。



目标是让网站负责整理材料，让 ChatGPT 负责深度分析和追问。



&#x20;



V5.3：数据状态栏



V5.3 阶段新增当前数据状态区块，用于提示摘要基于什么数据生成。



主要显示内容包括：



&#x20;本地摘要生成时间；

&#x20;市场价格状态；

&#x20;新闻缓存状态；

&#x20;最新新闻时间；

&#x20;最近 7 天相关新闻数；

&#x20;催化快照更新时间；

&#x20;催化状态；

&#x20;证据强度；

&#x20;数据新鲜度。



目标是解决“当前摘要基于什么时候的数据”的问题，提高研究结果的可解释性和可复核性。



&#x20;



&#x20;  当前项目定位



当前项目是一个本地 Streamlit 行业研究雷达工具，主要用于：



行业趋势初筛；

市场价格确认；

新闻催化跟踪；

催化框架复核；

本地研究摘要； 

ChatGPT 研究输入包导出；

数据状态提示。



本项目不提供直接交易指令，不构成投资建议，所有输出均应作为研究复核材料。
