# Resume Pitch

## A. 中文简历版

- 设计并迭代一个 AI-assisted 本地行业趋势研究系统，支持单行业搜索、行业雷达总览、市场价格确认、催化事件框架、轻量新闻观察和 Markdown / JSON 导出。
- 负责需求设计、功能拆解、投资研究逻辑、指标定义、行业数据口径、测试验证和迭代管理，并通过 Codex 辅助实现工程代码。
- 构建 V3 市场价格确认模块，使用 ETF / 龙头股代理样本计算 MA breadth、median 3M return、relative strength 和市场价格确认分。
- 建立 V4.1 全行业 catalyst framework 与 V4.2 轻量新闻观察层，支持本地缓存、去重、关键词匹配和 relevance 过滤。
- 新增多层 audit scripts，覆盖字段完整性、中文/英文一致性、news provider schema 和 semantic quality，提升研究数据可维护性。

## B. English Resume Version

- Designed and iterated an AI-assisted local industry trend research system with single-industry search, radar overview, market-price confirmation, catalyst framework, live-news watch, and Markdown / JSON export.
- Owned product requirements, feature decomposition, investment-research logic, metric definitions, data-quality rules, testing, and iteration management, with Codex assisting implementation.
- Built a V3 market-price confirmation layer using ETF and representative company proxies to compute MA breadth, median 3M return, relative strength, and market confirmation scores.
- Developed a V4.1 local catalyst framework and V4.2 lightweight news watch layer with caching, deduplication, keyword matching, and relevance filtering.
- Added audit scripts for data completeness, bilingual consistency, news-provider schema validation, and semantic-quality checks to improve maintainability and research reliability.

## Positioning Notes

This project should be described as a research assistant and product prototype, not as an automated trading or prediction system.

Recommended framing:

- AI-assisted development and iteration.
- User-led product design and research framework definition.
- Local-first data workflow.
- Explainable scoring and auditability.
- No brokerage connection and no direct trading instructions.
